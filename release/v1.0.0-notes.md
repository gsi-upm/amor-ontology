# AMOR ontology v1.0.0

AMOR v1.0.0 provides the reproducible ontology release accompanying the AMOR
paper.

## Included

- Normative AMOR, BHV, MFT, AMOR-BHV, AMOR-MFT, and compact example Turtle sources.
- Ten standalone competency-question SPARQL queries with traceability and expected bindings.
- Offline RDF syntax, model, example, competency-question, and generated-documentation validation.
- Reproducible WIDOCO 1.4.25 generation pinned by container digest.
- SHA-256 hashes for the normative paper-facing artefacts.

## Verification

The release notes must be completed with the verified final Git commit, public
representation check, and archived identifier before creating the GitHub
Release.
